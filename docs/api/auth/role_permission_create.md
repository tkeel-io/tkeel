### 角色权限添加
###### 接口功能
> 创建角色权限

###### HTTP请求方式
> POST
###### URL
>  /role/permission
###### 支持格式
> JSON


###### 请求参数
> |参数|必选|类型|说明|
> |:-----  |:-------|:-----|-----                               |
> |permission_type|true|body|权限维度{plugin、device}|
> |permission_id|true|body|实体ID{plugin_id、device_id}|
