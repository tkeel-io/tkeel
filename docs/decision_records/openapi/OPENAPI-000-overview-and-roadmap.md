# OPENAPI-000-overview-and-roadmap

## Overview
**OPENAPI** 是适用于平台插件的一套规范。这个规范定义了插件在平台的生命周期中与平台交互的几个 **HTTP** 接口的请求内容及返回值。

**OPENAPI** 满足 **HTTP** **通用规范**返回值。

后文中 `VERSION=v1`。
